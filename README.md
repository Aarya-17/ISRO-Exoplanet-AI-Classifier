# AI-enabled Detection of Exoplanets from Noisy Astronomical Light Curves

## Problem Statement
Develop an AI model to classify whether a detected transit candidate is a real exoplanet or a false positive.

## Role
AI & Classification Engineer

## Machine Learning Models
- XGBoost
- Convolutional Neural Network (CNN)

## Features Used
- Transit Depth
- Transit Duration
- Transit Period
- Signal-to-Noise Ratio (SNR)
- Even/Odd Transit Ratio
- Secondary Eclipse
- Noise Level
- Transit Shape

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## Files
- AI_Exoplanet_Classifier.ipynb
- synthetic_dataset.csv
- prediction_results.csv

## Output
Predicts whether a transit candidate is:
- Planet
- False Positive
